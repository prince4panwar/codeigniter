Arrays

In PHP, an array is a data structure that can hold multiple values. Arrays are one of the most versatile and commonly used data structures in PHP. They allow you to store and manipulate collections of data efficiently. PHP supports several types of arrays, including indexed arrays, associative arrays, and multidimensional arrays. Here's an overview of arrays in PHP:-

One Dimensional Array

$colors = array("Red", "Green", "Blue");//1D array
// or
$colors = ["Red", "Green", "Blue"];//1D array echo $colors[0]; echo $colors[1];

Assosiative array

$person = array(//Assosiative array
"first_name" => "ram",
"last_name" => "anuj",
"age" => 30
);
// or
$person = [//Assosiative array
"first_name" => "rohan",
"last_name" => "mohan",
"age" => 30
];
echo $person["first_name"];
echo $person["last_name"];

Two Dimensional Array

$employees = array(//2D array
array("John", "rohan", 30),
array("Jane", "Smith", 25),
array("Bob", "mohan", 40)
);
// or
$employees = [//2D array
["John", "Doe", 30],
["Jane", "Smith", 25],
["Bob", "Johnson", 40]
];
echo $employees[0][0];
echo $employees[1][2];

unset($fruits[0]); // Removes the element at index 0
// Create an indexed array
$fruits = ["Apple", "Banana", "Cherry", "Date", "Fig"];
// Use current() to get the current element
echo "Current Element: " . current($fruits) . "";
// Use next() to move the internal pointer to the next element
next($fruits); echo "Next Element: " . current($fruits) . "";
// Use end() to move the internal pointer to the last element
end($fruits);
echo "Last Element: " . current($fruits) . "";
// Use prev() to move the internal pointer to the previous element
prev($fruits);
echo "Previous Element: " . current($fruits) . "";
// Use reset() to move the internal pointer to the first element
reset($fruits);
echo "First Element: " . current($fruits) . "";
// Reset the pointer again
reset($fruits);
echo "After Reset: " . current($fruits) . "";
