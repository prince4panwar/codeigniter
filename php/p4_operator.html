Operators

Operators in PHP are symbols or special keywords that perform operations on variables and values. PHP supports a wide range of operators for tasks such as arithmetic, assignment, comparison, logical, and more. Here's an overview of the most commonly used operators in PHP :-

Here we discuss about scope of variable in php

1. Arithmetic Operator

echo " 5/2 = ",5/2;
echo " 5**2 = ",5**2;

4. Assignment Operator

echo "Assignment Operator (=, +=, -=, *=, /=, %=)";
$x = 10;
$y = 5;
$x = 10;
$x += 5; // Equivalent to $x = $x + 5;
echo $x; // Output: 15
$y = 20;
$y -= 7; // Equivalent to $y = $y - 7;
echo $y; // Output: 13
$a = 5;
$a *= 4; // Equivalent to $a = $a * 4;
echo $a; // Output: 20
$b = 50;
$b /= 10; // Equivalent to $b = $b / 10;
echo $b; // Output: 5
$c = 17;
$c %= 5; // Equivalent to $c = $c % 5;
echo $c; // Output: 2
$message = "Hello, ";
$name = "John";
$message .= $name; // Equivalent to $message = $message . $name;
echo $message; // Output: "Hello, John"
