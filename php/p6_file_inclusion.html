File Inclusion

In PHP, you can include the content of one file within another file using the include, require, include_once, and require_once statements. File inclusion is a common technique for reusing code, separating code into manageable files, and building modular applications. Here's an overview of these file inclusion methods:-

1. include Statement:

The include statement is used to include and evaluate a file. If the included file is not found, it generates a warning, but the script continues to execute. The include statement is often used for non-essential file inclusions, such as templates, libraries, or optional components.

2. require Statement:

The require statement is used to include and evaluate a file, just like include. However, if the included file is not found, it generates a fatal error and terminates the script.

// Example of using include
include 'my_file.php';
// Rest of your PHP code
echo "This code is after the include statement.";
// Example of using require
require 'my_file.php';
// Rest of your PHP code
echo "This code is after the require statement.";
