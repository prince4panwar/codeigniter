Scope of variable

In PHP, the scope of a variable refers to the part of the program where the variable can be accessed and used. PHP supports several types of variable scope, including global scope, local scope, and static scope. Here's an overview of each :-

1. Global Scope:

Variables declared outside of any function or class have global scope.

They can be accessed from anywhere in the PHP script, both inside and outside functions and classes.

2. Local Scope

Variables declared inside a function or method have local scope.

They are only accessible within the function or method where they are declared.

They do not exist outside of the function or method.
