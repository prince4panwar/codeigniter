String

In PHP, a string is a sequence of characters, such as letters, numbers, symbols, or spaces, enclosed in either single quotes (') or double quotes ("). PHP treats strings as a fundamental data type and provides a wide range of functions and operators for manipulating and working with strings. Here's an overview of strings in PHP :-

Define two strings for concatenation

// Define two strings for concatenation
$string1 = "Hello, ";//This is how we define the string
$string2 = "World!";//This is how we define the string

Concatenate the strings

// Concatenate the strings
$concatenatedString = $string1 . $string2;

Display the concatenated string

// Display the concatenated string
echo "Concatenated String: " . $concatenatedString . "";

Reverse the concatenated string

// Reverse the concatenated string
$reversedString = strrev($concatenatedString);
echo "Reversed String: " . $reversedString . "";

Find the position of "World" in the concatenated string

// Find the position of "World" in the concatenated string
$position = strpos($concatenatedString, "World");
if ($position !== false) {
echo "Position of 'World' in the string: " . $position . "";
} else {
echo "'World' not found in the string.";
}

Get the length of the concatenated string

// Get the length of the concatenated string
$length = strlen($concatenatedString);
echo "Length of the string: " . $length . "";

Convert the concatenated string to lowercase

// Convert the concatenated string to lowercase
$lowercaseString = strtolower($concatenatedString);
echo "Lowercase String: " . $lowercaseString . "";

Convert the concatenated string to uppercase

// Convert the concatenated string to uppercase
$uppercaseString = strtoupper($concatenatedString);
echo "Uppercase String: " . $uppercaseString . "";
