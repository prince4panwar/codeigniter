Data Types

In PHP, data types define the kind of data that can be stored in a variable. PHP supports several data types, including :-

1. Integer

2. Float

3. String

4. Boolean

5. Array

Here we discuss about all the data types of php

1. Integer :

Represents whole numbers, both positive and negative, without a decimal point. Example: $num = 42;

$a = 123;
$b = -1234;
echo "a = $a";
echo "a = $b";

2. Float :

Represents numbers with decimal points. Example: $floatNum = 3.14;

$c = 12.4;
echo "c = $c";

2. String :

Represents sequences of characters, enclosed in single or double quotes. Example: $str = "Hello, World!";

$d = "Hello World";
echo "d = $d";

3. Booleans :

Represents true or false values. Example: $isTrue = true;

$e = true;
$f = false;
echo "e = $e";
echo "f = $f";

4. Array :

Represents a collection of values. Arrays can be indexed (numeric or associative) or non-indexed.

$g = array(1,2,3,4,5);
var_dump($g);
echo "";

5. Null

$h = null;
var_dump($h);
