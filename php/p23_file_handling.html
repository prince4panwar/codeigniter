File Handling

File handling in PHP allows you to perform various operations on files, including reading, writing, creating, deleting, and more. PHP provides a set of functions and methods for file manipulation. Here's an overview of common file handling tasks in PHP :-

There are some steps of using file handling :-

1. Opening a File:

To work with a file, you must open it using the fopen() function.

fopen function takes two parameters: the file path and the mode For example:

"r" for reading

"w" for writing

"a" for appending

// Opening a File:
$file = fopen("example.txt", "r"); // Opens "example.txt" for reading.

2. Reading from a File:

If you opened the file in read mode, you can read its contents using functions like fgets(), fread(), or file_get_contents():

// Reading from a File:
$content = fread($file, filesize("example.txt")); // Reads the entire file content.

3. Writing to a File:

To write to a file, you need to open it in write mode ("w"):

// Writing to a File:
$file = fopen("output.txt", "w");
fwrite($file, "Hello, world!");

4. Appending to a File:

To append content to an existing file without overwriting it, use append mode ("a"):

// Appending to a File:
$file = fopen("existing_file.txt", "a");
fwrite($file, "Appending some text.");

5. Closing a File:

It's important to close the file when you're done to release system resources. Use the fclose() function:

// Closing a File:
fclose($file);
