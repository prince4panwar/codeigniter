Deleting data from the database

To delete data from a MySQL table using PHP, you need to use the SQL DELETE statement. Here's a step-by-step guide on how to delete data from a MySQL table using PHP :-

Prerequisites:

1. You need a web server with PHP installed.

2. You should have a MySQL database set up and the necessary credentials (username and password) to access it.

Steps to Create a Database in MySQL :

1. Connect to MySQL:

First, you need to connect to your MySQL server. You can do this using a MySQL client, phpMyAdmin, or the MySQL command-line client.

2. Delete Data from the Table :

Use SQL's DELETE statement to remove data from your table. Specify the table name and set the condition to identify the rows you want to delete.:

3. Execute SQL Queries:

With the connection established, you can execute SQL queries to interact with the database. Here's an example of querying a database:

Program :-

here we discuss everything about how do we delete data from the table of the database
$servername = "localhost";
$username = "root";
$password = "";
$database = "princedb1";
$conn = mysqli_connect($servername, $username, $password, $database);
if(!$conn)
{
die("We are having a problem. So we cannot connect with the database".mysqli_connect_error());
}
else
{
echo "We successfully connect with the database";
}
$sql = "DELETE FROM `employees` WHERE `Age` = '20' LIMIT 3";
$result = mysqli_query($conn, $sql);
$affected = mysqli_affected_rows($conn);
echo "Number of affected rows: $affected ";
if($result)
{
echo "Delete Successfully";
}
else
{
$error = mysqli_error($conn);
echo "Not Delete successfully due to this error ======> $error";
}

Output :-

We successfully connect with the database
Number of affected rows: 2
Delete Successfully
