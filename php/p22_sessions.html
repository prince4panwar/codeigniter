Sessions

Sessions in PHP allow you to store and manage user-specific data across multiple pages during a user's visit to a website. Sessions are a critical component of web applications for tasks like user authentication, shopping carts, and maintaining user data. Here's an overview of working with sessions in PHP :-

There are some steps of using Sessions :-

1. Starting a Session:

For using sessions, you have to first start the session

Session is now started and can store data

session_start();
// Your PHP code here

2. Storing Data in a Session

You can store data in the session by using the $_SESSION superglobal array.

// Storing Data in a Session:
$_SESSION['username'] = 'JohnDoe';
$_SESSION['user_id'] = 123;

3. Retrieving Data from a Session:

To retrieve data from the session, you simply access the $_SESSION superglobal array.

// Retrieving Data from a Session:
$username = $_SESSION['username'];
$user_id = $_SESSION['user_id'];

4. Unsetting a Session:

If you want to remove specific session data, you can use the unset() function.

// Unsetting and Destroying a Session:
unset($_SESSION['username']);

5. Destroying a Session

To end a user's session and clear all session data, you can use the session_destroy() function:

//session_destroy();

6. Checking for Session Existence:

To check if a session variable exists, you can use isset():

// Checking for Session Existence:
if (isset($_SESSION['username']))
{
echo "Session exist";
}
else
{
echo "Session doesn't exist";
}

Output :-

Session exist
