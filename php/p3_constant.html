Constants

In PHP, a constant is a variable that, once defined, cannot be changed or redefined during the execution of a script. Constants are used for storing values that should remain constant throughout the script's execution. Constants are case-sensitive by default and are typically defined in uppercase letters. To define a constant in PHP, you can use the define() function or the const keyword :-

Here we discuss about constant in php

1. Using define function :

define("str","Using define() function");
echo str;

1. Using const function :

const str2 = "Using const keyword";
echo str2;

Output :-

Using define() function
Using const keyword
