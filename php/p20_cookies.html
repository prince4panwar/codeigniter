Introduction of cookies

Cookies in PHP are small pieces of data that a web server sends to a user's web browser to be stored locally. Cookies are often used to track user information, such as session data or user preferences, between HTTP requests. In PHP, you can set, retrieve, and manipulate cookies using built-in functions. Here's an overview of working with cookies in PHP :-

1. Creating a cookie

Syntax: setcookie(name, value, expire, path, domain, secure, httponly);

$cookieName = "user";
$cookieValue = "John Doe";
$expirationTime = time() + 3600; // 1 hour from now
setcookie($cookieName, $cookieValue, $expirationTime);

2. Retrieving Cookie Values :

// Retrieving Cookie Values
if (isset($_COOKIE["user"])) {
$userName = $_COOKIE["user"];
echo "Welcome back, $userName!";
} else {
echo "Welcome to our website!";
}

3. Modifying cookies value

// Modifying cookies value
$newCookieValue = "Jane Doe";
setcookie($cookieName, $newCookieValue, $expirationTime);

4. Set the cookie's expiration time to a time in the past (e.g., 1 second ago)

setcookie($cookieName, "", time() - 1);

Output :-

Welcome to our website!
