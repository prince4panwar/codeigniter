Control Structure

Control structures in PHP are constructs used to control the flow of a PHP script. They determine how the program behaves based on conditional statements and loops. PHP provides a variety of control structures to handle different situations. Here are the main types of control structures in PHP :-

Control Structure

1. ifelse Statements

$temperature = 32;
if ($temperature > 30)//if statement
{
echo "It's a hot day!";
}
$temperature = 28;
if ($temperature > 30)//if else statement
{
echo "It's a hot day!";
} else {
echo "It's not too hot today.";
}
$grade = 75;
if ($grade >= 90) {//if else if else statement
echo "A";
} elseif ($grade >= 80) {
echo "B";
} elseif ($grade >= 70) {
echo "C";
} elseif ($grade >= 60) {
echo "D";
} else {
echo "F";
}

3. Ternary Operator

$age = 20;
// Using the ternary operator to determine if a person is a minor or an adult
$status = ($age < 18) ? "Minor" : "Adult";
echo "You are a $status.";
