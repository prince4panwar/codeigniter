Function

In PHP, a function is a reusable block of code that performs a specific task. Functions are an essential part of PHP programming and help organize and modularize your code. Here's how to define and use functions in PHP :-

Function definition with arguments

echo "FUNCTIONS";
// Function definition with arguments
function greet($name) {
echo "Hello, $name! (function with pararmeters)";
}

Function call with arguments

// Function call with arguments
greet("John");

Function definition without arguments

// Function definition without arguments
function sayHello() {
echo "Hello, World! (function without pararmeters)";
}

Function call without arguments

// Function call without arguments
sayHello();

Function definition with default argument

// Function definition with default argument
function greetDefault($name = "Guest") {
echo "Hello, $name! (function with default pararmeters)";
}

Function call without an argument (uses the default)

// Function call without an argument (uses the default)
greetDefault();

Two methods of passing arguments in php

1. Using call by value

echo "1. Using call by value";
function incrementValue($num) { $num++;
echo "Inside function: $num";
}
$number = 5;
incrementValue($number);
echo "Outside function: $number";

2. Using call by reference

echo "2. Using call by reference";
function incrementReference(&$num) {
$num++;
echo "Inside function: $num";
}
$number = 5;
incrementReference($number);
echo "Outside function: $number";

RECURSION

In which function calls itself again and again until it get the required result

function factorial($n) {
if ($n <= 1) {
return 1;
} else {
return $n * factorial($n - 1);
}
}
$number = 5; // Change this to the desired number for which you want to calculate the factorial. $result = factorial($number);
echo "Factorial of $number is $result";
